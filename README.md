Para compilar este projeto é necessário ter o libnds.
O arquivo .nds deve ser colocado no cartão SD de um R4, ou em cartão SD de um DSi com Twilight Menu, ou em seu emulador, na pasta de sua preferência.
Esta versão trás a tensão de não saber se você mesmo é o impostor.
Como jogar:
- Todas as pessoas receberão uma palavra, porém apenas uma pessoa receberá uma palavra diferente;
- O jogo decidirá quem começa a rodada, onde as pessoas terão que falar palavras relacionadas à palavra original;
- A quantidade de rodadas pode ser definida pelos próprios jogadores;
- No fim todos devem analisar e descobrir o impostor.
* As chances de ninguém ser o impostor são pequenas mas nunca zero.
